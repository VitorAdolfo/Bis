O que é o projeto?
==============
Alem de um doce sensacional, Bis é um bot simples criado por um desajeitado capaz de tocar músicas em canais de vóz do discord, permitindo uma festa eu suponho.

Como utilizar o bot?
====================
Para utiliar esse bot é um pouco simples, não sou o mestre da portabilidade, mas você deve baixar o projeto e criar um .env com as informações do seu bot, no mínimo dentro desse arquivo .env deve conter um parâmetro token
escrito assim:

BOT_TOKEN=Token_muito_longo_gerado_pelo_discord_028392u7

Para que seja possível controlar o seu bot, tirando isso basta execultar o arquivo main.py, ele ira criar um venv, baixar as bibliotecas necessárias para que o projeto possa funcionar e iniciar o arquivo bot.py que é
responsável pelas funções músicais do bot.

Considerações finais
====================
Sinta-se a vontade de usar editar completamente ou simplesmente fazer o seu bot do zero se esse projeto despertar alguma curiosidade, afinal de contas não sou nada mais do que um entusiasta da programação e esse projeto esta
longe de ser algo profissional.
